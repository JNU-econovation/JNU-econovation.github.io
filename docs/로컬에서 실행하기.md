# 로컬에서 실행하기

> main으로 merge 되기 전에 글을 확인하고 싶다면, 로컬에서 실행해보세요.

- 루비 설치

- jekyll 설치
  ```
  gem install jekyll bundler
  ```
- 저장소 clone
  ```
  git clone https://github.com/JNU-econovation/JNU-econovation.github.io.git
  ```
- 실행
  ```
  jekyll serve
  ```
  또는
  ```
  bundle exec jekyll serve
  ```
